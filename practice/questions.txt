Basic Questions
1. Write a Scala program to compute the sum of the two given integer values. If the two values are the same, then return triples their sum.  
2. Write a Scala program to get the absolute difference between n and 51. If n is greater than 51 return triple the absolute difference.  
3. Write a Scala program to check two given integers, and return true if one of them is 30 or if their sum is 30.  
4. Write a Scala program to remove the character in a given position of a given string. The given position will be in the range 0...string length -1 inclusive.  
5. Write a Scala program to find the larger value from two positive integer values in the range 20..30 inclusive, or return 0 if neither is in that range.
6. Obtain a temperature in degrees Fahrenheit from the user.  If the temperature is 80degrees or more display the message "Go play golf" otherwise, if the temperature is 70 -79 degrees display the message "Put on a jacket", otherwise display the message "It is way too cold."  Make a variable list, flowchart, and perform a desk check using the following values: 95, 72, 50.
7. Obtain a name and age from the user.  If the user is 16 or older, output a message indicating they are old enough to drive.  For people under 16, output a message indicating how many years they must wait before they can drive legally. 
8. Check if a given number is a palindrome. 
9. Check if a given number is an Armstrong number.
